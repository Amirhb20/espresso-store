# ☕ Espresso Store

A modern and responsive espresso e-commerce website built with **HTML**, **Tailwind CSS**, **JavaScript**, and **Swiper.js**.

## 🚀 Technologies

- HTML5
- Tailwind CSS
- JavaScript (ES6+)
- Swiper.js

## ✨ Features

- Modern and clean UI
- Fully responsive design
- Interactive product slider
- Smooth animations
- Mobile-first layout
- Reusable components
- Optimized performance

## 🛠️ Installation

Clone the repository:

```bash
git clone https://github.com/your-username/espresso-store.git
```

Go to the project folder:

```bash
cd espresso-store
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```
